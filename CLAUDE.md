# openalex_manage

## Project Overview
Management and analysis of OpenAlex data.

## Directory Structure
```
openalex_manage/
├── CLAUDE.md
├── claude_codes/
│   ├── codes/          # .py and .R scripts + .sh SLURM jobs
│   ├── notebooks/      # .ipynb for interactive exploration
│   ├── figures/        # saved plots, organized by date subfolder
│   ├── plans/          # plan files
│   └── findings/       # findings after analysis
└── data/
    ├── intermediates/  # intermediate outputs
    └── schemas/        # schema logs
```
